Node.js 简介

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境，它允许开发者使用 JavaScript 编写后端代码，具有非阻塞 I/O、事件驱动等特性，适合高并发应用场景。

  • 官方网站：https://nodejs.org
  • 当前主流版本支持 ES6+、CommonJS、部分 ES 模块。

全局对象

Node.js 提供了一些可以在任意模块中直接访问的 全局变量/对象，不需要 require 引入。以下是一些重要的全局对象：

1. global

global 是 Node.js 中的全局作用域对象，类似于浏览器中的 window 对象。

2. process

process 对象提供了与当前 Node.js 进程相关的信息和控制。

3. console

console 对象提供标准输出、错误输出的接口。

4. __dirname 和 __filename

__dirname 表示当前模块文件所在目录的绝对路径，__filename 表示当前模块文件的绝对路径的文件名。

5. setTimeout, setInterval, setImmediate

全局定时器函数，用于处理定时器和延迟执行。

基本内置模块

Node.js 提供了许多内置模块，以下是一些常用的模块：

1. path

path 模块用于处理文件路径。常用方法有：

方法 说明
path.join() 拼接路径
path.resolve() 解析为绝对路径
path.basename() 获取文件名
path.dirname() 获取目录名
path.extname() 获取扩展名

2. os

os 模块提供了与操作系统相关的实用方法和属性。

3. url

url 模块用于 URL 解析和处理。

4. querystring

querystring 模块用于解析和格式化 URL 查询字符串。
